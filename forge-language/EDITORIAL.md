# Forge Editorial Policy

## Document Template
Every FLS document SHALL use the standard template unless superseded by an approved RFC.

## Identifier Policy
- Sections: §N
- Requirement IDs: FLS-XXX-YYY-NNN
- Example IDs: EX-...
- Figure IDs: FIG-...
- Table IDs: TAB-...

Stable requirement IDs SHALL NOT be renumbered after publication.

## RFC Keywords
Normative requirements SHALL use RFC 2119 terminology.

## Review Checklist
- Normative terms defined
- Stable requirement IDs
- Testable requirements
- No implementation details in normative sections
